PROJECT TITLE
AI Trading Signal Platform (Production-Grade Fintech Microservices System)

OBJECTIVE
Build a professional fintech trading signal platform that generates automated buy/sell signals from market data and displays them in a modern trading dashboard similar to TradingView or Binance.

The system must use a scalable microservices architecture with a NestJS API gateway and a Go-based signal engine.

The project should be production-ready with Docker containerization, Kubernetes scaling, CI/CD automation, and a clean GitHub-ready structure.
