## Packages
recharts | Required for the interactive financial charts (candlestick/area mocks)
date-fns | Required for formatting timestamps on the signal feed
lucide-react | Required for dashboard icons

## Notes
- Using a "dark mode default" fintech theme with deep blues and neon accents.
- Simulating live websocket signal feed by using a frontend `setInterval` that periodically creates new signals via the API.
- Using `JetBrains Mono` for numerical data to provide a Bloomberg Terminal / Pro Trading feel.
